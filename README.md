# eazy-sms
This library aims to provide an easy way to send sms 
using mnotify, hubtel sms gateway , etc.. in your .netcore applications.

###### SMS Providers
- [ ] Mnotify
- [ ] Hubtel

###### SMS Channels
- SMS
- Mail
